How it works

Use this extension to send selected text from any page or PDF viewer to your NotyPDF server.

You can also right-click a selection and choose Send selection to NotyPDF to send it immediately.

Configuration

  • Open Settings to set the server URL.
  • If the server is exposed through a reverse proxy such as Nginx Proxy Manager, enter the external URL and any authentication credentials.
  • Provide username and password if your proxy requires them.

PDF Selection

When viewing a PDF, copy the text to the clipboard before sending. Chrome cannot directly read selections inside PDFs.

Note

If running NotyPDF in a local Docker container, connect to http://localhost:5026 instead of http://localhost:3000.
